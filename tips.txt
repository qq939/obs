# VSCode Mac Shortcuts & Tips

## General
- **Command Palette**: `Cmd + Shift + P`
- **Quick Open**: `Cmd + P`
- **Toggle Sidebar**: `Cmd + B`
- **Toggle Terminal**: `Ctrl + ` `

## Editing
- **Multi-cursor**: `Opt + Click` or `Cmd + Opt + Up/Down`
- **Move Line Up/Down**: `Opt + Up/Down`
- **Copy Line Up/Down**: `Shift + Opt + Up/Down`
- **Delete Line**: `Cmd + Shift + K`
- **Comment/Uncomment**: `Cmd + /`
- **Rename Symbol**: `F2` (Refactor rename variables/functions globally)

## Intelligent Code Completion (IntelliSense)
- **Trigger Suggestion**: `Ctrl + Space` (Manually trigger completion list)
- **Accept Suggestion**: `Tab` or `Enter` (When a suggestion is highlighted)
- **Snippet Expansion**: Type a few letters (e.g., `def`, `for`, `class`) and press `Tab` to expand standard templates.
- **Parameter Hints**: `Cmd + Shift + Space` (Show arguments for function call)
- **Quick Fix**: `Cmd + .` (Auto-import, fix errors)
- **Go to Definition**: `F12` or `Cmd + Click`
- **Peek Definition**: `Opt + F12`

## Python Specific
- **Run Python File**: `Run Button` (top right) or setup launch.json
- **Format Document**: `Shift + Opt + F`
- **Organize Imports**: `Shift + Opt + O`
