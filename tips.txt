# VS Code Tips for Mac

1. **IntelliSense / Code Completion**:
   - Trigger suggestion: `Ctrl + Space` (or just type and wait)
   - Accept suggestion: `Tab` or `Enter`

2. **Navigation**:
   - Go to file: `Cmd + P`
   - Go to symbol: `Cmd + Shift + O`
   - Go to definition: `F12` or `Cmd + Click`

3. **Editing**:
   - Comment/Uncomment line: `Cmd + /`
   - Multi-cursor: `Option + Click`
   - Move line up/down: `Option + Up/Down`
   - Copy line up/down: `Shift + Option + Up/Down`

4. **Terminal**:
   - Toggle terminal: `` Ctrl + ` ``
